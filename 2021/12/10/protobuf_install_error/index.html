protobuf安装以及一些问题解决

Author: Space Date: 十二月 10, 2021  0:00:00 Category: Linux

protobuf安装以及一些问题

— ↓按照以下命令执行↓ —

    •  sudo apt-get install git

    •  git clone https://github.com/protocolbuffers/protobuf.git 

    •   cd protobuf 

    •   git submodule update --init --recursive

    •   ./autogen.sh 

    • ./configure 

    •    make 

    •  make make check 

    •  sudo make install

    •  sudo ldconfig

错误解决

下载protobuf报错 443 或者 其余端口
解决方法: 换手机热点进行下载，科学上网进行下载， 多试几次。

初始化git报错
解决方法: 先执行git init 在执行git submodule update –init –recursive

执行 make check报错
报错1.

解决方法： 在虚拟机->编辑虚拟机设置->内存更改为8GB即可解决

成功图片:

错误2：

解决方案: 暂无，这个并不会影响生成的.h和.cc文件

安装完成后将代码写好 后缀名为.proto文件.
执行:
protoc -I=./ –cpp_out=./ 文件名.proto
之后就会生成.h和.cc文件
